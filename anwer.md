# Answers Document

## 1. Project Overview

This project is a Pomodoro Timer built using HTML, CSS, and JavaScript. It demonstrates DOM manipulation, timer logic, and event handling.

## 2. Assumptions

* Standard Pomodoro technique (25 min work, 5 min break)
* Runs entirely in browser (no backend)
* No external libraries used

## 3. Approach

* Used JavaScript `setInterval()` for countdown timer
* Managed state using variables (time, isRunning, mode)
* Updated UI dynamically using DOM methods
* Handled start, pause, reset functionality

## 4. Challenges Faced

* Managing pause/resume without resetting timer
* Switching between work and break sessions automatically
* Keeping UI and timer state synchronized

## 5. Improvements (Future Scope)

* Add user-defined timer settings
* Store session history using localStorage
* Add sound notifications
* Improve UI with animations and themes

## 6. Key Learnings

* Improved understanding of JavaScript timing functions
* Better DOM manipulation skills
* Learned basic state management in vanilla JS

## 7. Conclusion

This project helped in understanding how frontend logic works in real-time applications like timers.
