# Pomodoro Timer

## Overview

This is a simple Pomodoro Timer application built for a frontend assessment. It helps users manage study/work sessions using the Pomodoro technique (25 minutes work, short breaks).

## Features

* Start, pause, and reset timer
* Work session and break switching
* Clean and responsive UI
* Simple vanilla JavaScript implementation (or your chosen stack)

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the project
2. Open the project folder
3. Open `index.html` in your browser

OR

If using a local server:

```bash
npx serve
```

Then open the provided local URL in browser.
